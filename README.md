# � Spotlight - 智能点名助手

一款功能强大、界面美观的桌面端课堂随机点名工具，专为教师设计。

## ✨ 核心特性

### 🎲 抽取功能

- 单人/多人随机抽取
- 炫酷抽取动画效果
- 音效反馈（可关闭）
- **加权随机**：被抽中次数少的学生概率更高
- **冷却机制**：刚抽中的学生 N 次内不再参与

### 📚 班级与学生管理

- 多班级管理，数据隔离
- Excel / TXT 批量导入学生
- 批量照片导入（支持学号/姓名匹配）
- 智能导入预览，自动识别无效数据

### 🏆 积分系统

- 抽中后可直接加减分
- 积分排行榜页面
- 积分导出 Excel

### 📊 数据统计

- 抽取历史记录
- 柱状图可视化
- 历史导出 (Excel/TXT)

### 🎨 主题系统

- 6 种 UI 风格：Material / Fluent / Apple / Flat / Neumorphism / Glassmorphism
- 浅色/深色模式 + 跟随系统

### 🔒 安全

- 密码保护
- 一键备份与还原 (Zip)

### 🌐 国际化

- 简体中文 / English

---

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发模式
npm start

# 打包 Windows 版本
npm run build:win
```

## 🛠️ 技术栈

- **Electron** + HTML/CSS/JS（无框架）
- **Chart.js** 数据可视化
- **本地 JSON** 数据存储
